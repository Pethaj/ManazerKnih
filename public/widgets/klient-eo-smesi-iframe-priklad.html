🌿 EO Směsi Chat - Embedding na Bewit.cz

Kompletní průvodce implementací chatu o esenciálních olejích na vašem webu

🎯 Co tento chat dělá

Odpovídá na dotazy o esenciálních olejích a směsích BEWIT
Automaticky sleduje uživatelská data (ID, email, jméno, e-shop token)
Ukládá historii konverzací do databáze
Plně responzivní - funguje na mobilu i desktopu
Žádná údržba - aktualizace automaticky z MedBase

📋 Implementace - Krok za krokem

1. Otevřete šablonu vašeho webu (např. footer.php, template.php)

2. Vložte tento kód před uzavírací tag </body>

<!-- ============================================= --> <!-- EO SMĚSI CHAT WIDGET - START --> <!-- ============================================= --> <!-- Iframe s chatem --> <iframe id="eo-smesi-chat-iframe" src="https://gr8learn.eu/embed-eo-smesi.html" <!-- 👤 User data (automaticky se sbírají) --> data-user-id="<?php echo $user->id; ?>" data-firstname="<?php echo $user->firstName; ?>" data-lastname="<?php echo $user->lastName; ?>" data-email="<?php echo $user->email; ?>" data-position="<?php echo $user->position; ?>" data-token-eshop="<?php echo $user->bewitToken; ?>" data-cc-potential-raynet="<?php echo $user->ccPotentialRaynet; ?>" <!-- 🎨 Styling --> style="position:fixed; right:24px; bottom:24px; width:1200px; height:700px; max-width:calc(100vw - 48px); max-height:calc(100vh - 48px); border:0; border-radius:16px; box-shadow:0 25px 50px -12px rgba(0,0,0,0.25); z-index:999999;" <!-- ⚙️ Permissions --> allow="clipboard-write" title="EO Směsi Chat" ></iframe> <!-- 📤 JavaScript pro odesílání user dat (volitelné) --> <script> (function() { const iframe = document.getElementById('eo-smesi-chat-iframe'); if (!iframe) return; // Počkej na načtení iframe iframe.addEventListener('load', function() { // Pošli user data do chatu iframe.contentWindow.postMessage({ type: 'USER_DATA', user: { id: iframe.getAttribute('data-user-id'), firstName: iframe.getAttribute('data-firstname'), lastName: iframe.getAttribute('data-lastname'), email: iframe.getAttribute('data-email'), position: iframe.getAttribute('data-position'), tokenEshop: iframe.getAttribute('data-token-eshop'), cc_potential_raynet: iframe.getAttribute('data-cc-potential-raynet') } }, 'https://gr8learn.eu'); }); })(); </script> <!-- ============================================= --> <!-- EO SMĚSI CHAT WIDGET - END --> <!-- ============================================= -->

👤 Mapování User Data Fieldu

Ujistěte se, že mapujete správné PHP proměnné z vašeho systému:

Data atribut PHP proměnná (příklad) Popis
data-user-id $user->id ID uživatele v Bewit DB
data-firstname $user->firstName Křestní jméno
data-lastname $user->lastName Příjmení
data-email $user->email Email
data-position $user->position Pozice/role
data-token-eshop $user->bewitToken ⚠️ DŮLEŽITÉ - E-shop token

⚠️ DŮLEŽITÉ - E-shop Token

Pole data-token-eshop je kritické pro integraci s Bewit e-shopem!

  • Tento token propojuje uživatele v chatu s jeho účtem v e-shopu
  • Bez tokenu nebude fungovat doporučení produktů a nákupní funkce
  • Ověřte že máte správné jméno PHP proměnné v Bewit systému
  • Token se ukládá do databáze spolu s chat historií

V MedBase admin panelu vidíte nové zprávy v historii

🔍 Debugging

Pokud chat nefunguje:

  1. Otevřete Developer Console (F12)
  2. Hledejte logy začínající "🔥 EMBED EO SMESI CHAT"
  3. Zkontrolujte že vidíte: "📋 User data načtena z data-* atributů iframe"
  4. Ověřte že všechny PHP proměnné existují (žádné PHP errors)
  5. Zkontrolujte že URL https://gr8learn.eu/embed-eo-smesi.html je dostupná

📊 Co se děje s daty

Všechna user data se automaticky:

  • ✅ Ukládají do Supabase databáze v tabulce chat_messages
  • ✅ Posílají do N8N webhooku pro zpracování AI
  • ✅ Jsou dostupné v admin panelu MedBase
  • ✅ Lze filtrovat podle emailu, ID, nebo tokenu
  • ⚠️ Nejsou šifrovaná - nepoužívat pro hesla nebo platební údaje
